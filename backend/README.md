## Quick Start

```bash
# Install dependencies
npm install

# Setup environment
cp env.example .env
# Edit .env with your credentials

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Tech Stack

- Node.js + Express
- TypeScript
- MongoDB + Mongoose
- JWT Authentication
- Razorpay (Payments)
- Cloudinary (Images)
- Mailgun (Emails)

## API Endpoint

Development: `http://localhost:5002/api/v1`
